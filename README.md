# Dotfiles + Neovim + Zsh Installer

This repository contains configuration files and an automated setup script that installs:

- **Zsh**
- **Oh-My-Zsh**
- **Ripgrep**
- **Neovim v0.11.5** (built from source)
- **Build tools (CMake, Ninja, etc.)**
- A complete **Neovim config**
- Custom **Zsh config and themes**

It supports:

- **Arch Linux**
- **Debian / Ubuntu-based systems**

---

## Installation

```bash
git clone https://github.com/kaylukes/dotfiles.git
cd dotfiles
chmod +x install.sh
./install.sh
```
